Projects

LED Album Art Display

This started as a side project to make something to decorate my room, but grew into more when I enrolled in an embedded design class involving the creation of a custom PCB. The finished project displays album art and allows volume and playback control via a rotarty encoder. All code was written by me utilizing an STM32 MCU and ESP32. Source code can be found on my github and you can check out a demo video here.

FPGA Video Game

I developed a retro racing game on a Spartan-7 FPGA using SystemVerilog and C, showcasing expertise in hardware-software co-design. This involved creating a custom CPU, I/O peripherals, memory management, and GPU logic to simulate real-time graphics. Source code can be found on my github and you can check out a demo video here.
